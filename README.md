## Getting Started

### Prerequisites

- Node.js 18+  
- OpenAI API key  

### Installation

```bash
# Create Next.js project (if starting fresh)
npx create-next-app@latest ai-chatbot
cd ai-chatbot

# Install dependencies
npm install langchain openai

## Run the development server:
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
